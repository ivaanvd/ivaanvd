## 🛠️ Mis Proyectos Principales

### 🗺️ GeoAda - Gestión Inteligente de Espacios
*Sistema para la visualización y reserva de salas en el edificio Ada Byron (Unizar).*

**Tecnologías:** `Node.js` `React` `Prisma` `MongoDB` `PyGeoAPI` `Arquitectura Hexagonal`

- **Funcionalidades clave:**
    - Mapa interactivo por plantas para localizar aulas, laboratorios y despachos.
    - Sistema de validación de reservas basado en roles (estudiantes, docentes, gerencia).
    - Implementación de **Diseño Guiado por el Dominio (DDD)** para asegurar la escalabilidad.
- **Lo que aprendí:** Manejo de datos espaciales y estructuración de proyectos de software complejos siguiendo estándares de industria.

> **Estado actual de la app:** Prototipo funcional / Fase de pruebas de arquitectura.

### 🚨 Proyecto Montgomery - Crime Analytics Portal
*Aplicación web para el análisis y visualización de criminalidad en el Condado de Montgomery.*

**Tecnologías:** `React` `OpenStreetMap` `OSRM` `Análisis de Datos`

- **Funcionalidades clave:**
    - Panel centralizado para la policía con visualización de delitos geolocalizados.
    - Cálculo de **rutas más seguras** para ciudadanos mediante el índice IPP.
    - Mapas interactivos con filtros avanzados por tipo de delito y distritos.
- **Lo que aprendí:** Integración de APIs de mapas, visualización de grandes volúmenes de datos y diseño orientado a la seguridad ciudadana.

> **Estado actual de la app:** En desarrollo (Implementando modelos de predicción y alertas).

### 🤖 Asistente Sanbot - Robótica Social
*Interfaz asistencial para el robot humanoide Sanbot orientada a personas mayores.*

**Tecnologías:** `Android Studio` `Java/Kotlin` `Hardware Robótico` `UX Accesible`

- **Funcionalidades clave:**
    - Desarrollo de la interfaz nativa desplegada directamente en el robot.

> **Estado actual de la app:** Fase de pruebas de usuario y validación de metodología.
